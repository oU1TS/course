# Separating Content and Adding Description Links - Deep Linking & Multiple Lectures

This plan details the changes required to support multiple related lecture IDs on resources (using a selection popup modal) and deep-linking/sharing buttons that copy custom target URLs to the clipboard.

## Proposed Changes

---

### [Component: Data Updates & ID Mapping]

#### [MODIFY] [resource.json](ocuments/GitHub/[oU1TS]/course/resource.json)
- Add unique `resourceId` to each item inside the resources array.
- Update `"relatedLectureId"` to hold multiple lecture IDs (e.g. array of strings `["cse-211-oop-l1", "cse-211-oop-l2"]`) or keep single strings where applicable.

---

### [Component: UI and Styling Updates]

#### [MODIFY] [index.html](ocuments/GitHub/[oU1TS]/course/index.html)
- Add a new selection modal overlay `#lecture-select-modal` below the existing `#roadmap-modal`. This modal will display a list of related lectures.

#### [MODIFY] [style.css](ocuments/GitHub/[oU1TS]/course/style.css)
- **Select Modal styles**: Style the lecture buttons inside the select modal (`.select-modal-link`, `.lecture-name`, `.course-name-sub`).
- **Copy Button styles**: Style the copy icon button (`.btn-copy-id`, `.copy-icon`, `.copy-icon.success`, `.btn-copy-id.copy-success`).

---

### [Component: Client SPA Controller]

#### [MODIFY] [app.js](ocuments/GitHub/[oU1TS]/course/app.js)
- **Deep Linking in Routing**:
  - In `routePage()`, parse `queryParams.resource` when loading `#resources`.
  - In `setupViewInteractions()`, handle scrolling to the resource card and flashing it if `queryParams.resource` matches.
- **Copy to Clipboard Engine**:
  - Add helper function `copyLinkToClipboard(type, id, button)` to write deep links to the clipboard and temporarily transition the button icon to a green checkmark on success.
- **Related Lecture Select Modal Controllers**:
  - Add `openLectureSelectModal(lectureIds)` and `closeLectureSelectModal()`. If lectures data is not loaded/cached, fetch `lecture.json` dynamically first, show a loading skeleton, then build the list of lecture titles.
- **Render Functions**:
  - Update `Render.discussions()` to output the copy button next to each course title (in the accordion header row) and lecture card.
  - Update `Render.resources()` to support `resourceId`, multiple `relatedLectureId` values, and output the copy button next to each resource title. If multiple IDs are present, render a click handler triggering `openLectureSelectModal`.
- **View Interactions**:
  - In `setupViewInteractions()`, bind the copy buttons click handlers (stopping propagation) and wire the "View Related Lectures" multiple select trigger.
