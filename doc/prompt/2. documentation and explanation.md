### Clarification on the Hero Section Loading Code:
As detailed in the new reference guide, the Hero section is **not** statically defined in `index.html`. It is generated dynamically inside `renderHome(data)` in **[render.js](Documents/GitHub/[oU1TS]/course/render.js#L20-L81)**:
1. **Source Variable**: The tagline underneath the title is populated using the top-level `"motto"` key inside `data.json`.
2. **Template Compilation**: `render.js` reads `data.motto`, escapes it to prevent XSS, and binds it to a paragraph tag with the class `hero-motto`:
   ```javascript
   // render.js
   const motto = data.motto || '';
   // ...
   return `
     <div class="hero-wrapper">
       <h1 class="hero-title">UITS <br><span class="accent-text">Course Mastery</span></h1>
       <p class="hero-motto">${escapeHTML(motto)}</p>
       ...
   `;
   ```
3. **Mounting**: When the URL hash is `#home`, the router in [app.js](Documents/GitHub/[oU1TS]/course/app.js) executes `window.Render.home(appState)` and mounts the generated HTML directly into `<main id="content-app">`.
