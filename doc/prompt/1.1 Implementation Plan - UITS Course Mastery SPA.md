# Implementation Plan - UITS Course Mastery SPA

We will build the **UITS_Course_Mastery** website as a single-page application (SPA) under the oU1TS banner. The website uses vanilla HTML, CSS, and JS, featuring dynamic template rendering, theme switching, responsive design, and state-driven configuration.

## Proposed Architecture

```mermaid
graph TD
    index.html --> app.js
    app.js -->|fetches| data.json
    app.js -->|intercepts Hash / Nav| Router
    Router -->|updates| render.js
    render.js -->|renders templates| content-app["#content-app Viewport"]
    style.css -->|styles| index.html
```

## Proposed Changes

We will create the following files in the repository root:

### Core SPA Components

#### [NEW] [index.html](Documents/GitHub/[oU1TS]/course/index.html)
- Main HTML shell containing Google fonts (Outfit, Inter), favicon, CSS links, page header with logo, navigation links, and theme toggle switch.
- Contains the hamburger button for mobile viewports.
- Contains the target viewport: `<main id="content-app"></main>`.
- Contains the Modal Container skeleton for the roadmap popup overlays.

#### [NEW] [style.css](Documents/GitHub/[oU1TS]/course/style.css)
- Pure black/white themes:
  - Dark Theme (default): Background is pure black (`#000000`), text is pure white (`#ffffff`).
  - Light Theme: Background is pure white (`#ffffff`), text is pure black (`#000000`).
  - Minimalist design: No gradients. Uses clean borders, solid colors, and subtle accents.
- CSS Variables for color palette, borders, shadows, transitions, and responsive grid layouts.
- Header, navigation, mobile hamburger drawer transitions.
- Interactive styling for the roadmap timeline blocks and active modal overlays.
- Custom skeleton loaders for asynchronous fetching states.

#### [NEW] [render.js](Documents/GitHub/[oU1TS]/course/render.js)
- UI Template rendering functions that return HTML strings or build DOM fragments based on the loaded state.
- `renderHome(data)`: Renders the motto and the 5-step interactive roadmap block diagram.
- `renderDiscussions(data)`: Renders the recorded lectures grid with course names, semester tags, video buttons, and notes buttons.
- `renderResources(data)`: Renders categorized toolkits and academic tracks.
- `renderAbout(data)`: Renders UITS Course Mastery vision, core points, and parent oU1TS portal links.
- `renderJoin(data)`: Renders onboarding step-by-step guidance for FB and WhatsApp groups.

#### [NEW] [app.js](Documents/GitHub/[oU1TS]/course/app.js)
- Application controller managing initialization, fetching `data.json`, router hash intercepts, theme toggle toggler, mobile menu drawer toggle, and roadmap modal controllers.
- Intercepts clicks or hash changes, clears `#content-app`, and dynamically injects templates from `render.js`.

#### [NEW] [data.json](Documents/GitHub/[oU1TS]/course/data.json)
- Centralized data schema containing all the motto texts, roadmap step definitions (summary, deep purpose), discussions recordings, resources listings, about details, and onboarding instructions.
